# Calculator 

## Description

This is a simple calculator that performs basic and advanced mathematical operatins.

## Features

 = Addition (+)
 = Subtraction (-)
 = Multiplication (*)
 = Division (/)
 = Modulus (%)
 = Power 
 = Square
 = Cube
 = Input validation using 'try-execpt'
 = Input operation validation
 = Division by Zero handling
 = Continue or exit option ('yes/no')

## Technologies Used

 = Python3

## How to Run 

1. Download or clone this repository.
2. Open the project in VS Code.
3. Run the Python file.
4. Choose an operation.
5. Enter the required number(s).
6. View the result.
7. Enter `yes` to perform another calculation or `no` to exit.
